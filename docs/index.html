About

Project Overview

Early detection of breast cancer significantly improves patient survival rates. This project investigates whether classical machine learning classifiers — trained on deep visual features extracted from a pretrained convolutional neural network — can reliably distinguish malignant from benign tumors in breast ultrasound images.

Transfer Learning

VGG16 pretrained on ImageNet extracts 25,088-dimensional spatial features from each image without training from scratch.

Dimensionality Reduction

PCA compresses features from 25,088 to 70 components, addressing overfitting while retaining discriminative structure.

Clinical Threshold Tuning

Lowering the decision threshold to 0.3 raises malignant recall from 80% to 89%, reducing the cost of missed diagnoses.

Unsupervised Clustering

ResNet50 features fed into K-Means (k=2, k=3) reveal natural image groupings without the use of ground-truth labels.

Methodology

Project Pipeline

A four-stage workflow from raw ultrasound images to evaluated classifiers.

1

Data Exploration

Inspected image counts (437 benign, 210 malignant), verified class distribution, and examined image dimensions across the dataset.

01-data-exploration.ipynb
2

Feature Extraction

VGG16 (ImageNet weights, classification head removed) processes each 224×224 image into a 7×7×512 feature map, saved as .npy arrays.

02-data-processing.ipynb
3

Supervised Training

StandardScaler → PCA(70) → 80/20 split → Logistic Regression, KNN, and SVM trained and evaluated with confusion matrices, ROC curves, and AUC.

03-model-training.ipynb
4

Unsupervised Clustering

ResNet50 extracts 2048-dim features per image. K-Means with k=2 and k=3 clusters images; PCA (2 components) visualizes cluster separation.

04-unsupervised-model.ipynb

Clinical note — In a medical screening context, failing to detect a malignant case carries far greater risk than a false positive. The threshold-0.3 Logistic Regression achieves the highest malignant recall (88.6%) at the cost of overall accuracy, making it the most clinically conservative choice among the models evaluated.
